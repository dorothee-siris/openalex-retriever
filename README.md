\# OpenAlex Institution Publications Retriever



A Streamlit application for retrieving publication metadata from OpenAlex for selected institutions.



\## Features

\- Search and select institutions from a curated database

\- Configure publication retrieval parameters (timeframe, document types, metadata fields)

\- Export results as CSV with deduplication



\## Usage

1\. Search for institutions by name, acronym, or alternative names

2\. Select up to 10 institutions

3\. Configure retrieval parameters

4\. Download the results as CSV
